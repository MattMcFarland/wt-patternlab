# Build

## IMPORTANT! PHP REQUIRED

Once PHP is installed...

```
npm install
```
```
npm start
```

# Serve up
You can serve up static using public/index.html

Or you can use gulp
```
gulp serve
```

Or npm
```
npm start
```
